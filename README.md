# Titanic Survival Prediction

## Project Overview

This project predicts passenger survival on the Titanic using Machine Learning. It includes data cleaning, exploratory data analysis (EDA), feature engineering, Logistic Regression model building, evaluation, and Kaggle competition submission.

## Dataset

- Source: Kaggle - Titanic: Machine Learning from Disaster
- Training Records: 891
- Test Records: 418

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow

- Data Loading
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Data Encoding
- Train-Test Split
- Logistic Regression
- Model Evaluation
- Kaggle Submission

## Model Performance

- Model: Logistic Regression
- Test Accuracy: **81.01%**
- Kaggle Public Score: **0.76315**
